när programet startar kan du skapa ett konto och sedan logga in med det.
 Dock går det inte att testa alla funktioner när du är vanligt inloggad.
 För att komma åt funktionerna för att visa alla customers och ta bort en customer så måste man logga in med ett konto som har användarnamnet admin.
 Om du använder min json fil när du kör programet finns adminkontot redan med lössenordet 1234.
 Anars kan du bara skapa ett konto med namnet admin och det kommer också funka. 

 För att spara det du har gjort i programet måste du gå tillbaka till startmenyn.
 Det gör du genom att antingen trycka 3 eller 7 beroende på om du är inloggad som admin eller en vanlig användare.
 Och efter det trycker du 3 för att spara och avsluta programet.
